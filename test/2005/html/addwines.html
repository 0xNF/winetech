As you build your cellar you may find wines you own that are not contained in the database. These can easily be added by copying the most similar wine found in the database and, typically, just adding a new vintage. If necessary, new producers, new wines and any other new information can be entered.

Wines, once they are in the database, are added to the visual cellar by finding them in the database, indicating the number of bottles, and "dragging" them to the appropriate location where they will "drop" into the cellar bins you indicate. This process is shown above. It involves three windows used together:

  • The cellar window into which the wines will be placed (upper left)
  • A control window (lower left)
  • A window listing selected wines from the database (right)
In this example, the "Find Wines to Add" button in the control window has been pressed. This brings up a wine search form. "1990 Bordeauxs" were specified and a list of all such wines in the database was created, as shown.

Next, the wine to be added is found in this list and the "number of bottles to add" is typed (in this case "2" bottles of 1990 l'Angelus). Finally, the mouse is dragged from the area next to the bottle count into the proper starting bin in the cellar. Two bottles will be automatically placed into the cellar.
