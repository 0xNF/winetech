Viewing all Tastings for a Wine at One Time
The Main Form provides a convenient vehicle for paging through the tastings for any particular wine. As an alternative, you can see them all at once using < Edit > < Tastings > while looking at either a Table of wines or the Main Form itself. As shown below, this gives you an "at-a-glance" overview of all Parker tasting -- plus any of your own.

This illustration contains only a subset of the columns available with each tasting. Also, the entire tasting note is not shown. You could enlarge the window to show all of this data at once.
