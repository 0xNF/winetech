Robert Parker's Wine Advisor and Cellar Manager Overview

  • An electronic wine database built from past (and future) issues of The Wine Advocate ®

  • Tools for tracking wines of particular interest to you or managing a wine shopping list
  • The capability to import wines from your previous software system
