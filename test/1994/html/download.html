Test drive Robert Parker's Wine Advisor and Cellar Manager. You can download the demo version of our software right now. Click here for the hardware and software requirements to run the full program. Disk space requirments for the demo are much less and can be found below. It is fully functional except for the following:
  • Only about 130 wines (those that are used in our Sample Cellars) are included in the sample wine database.
  • All of Robert Parker's tastings from the years 1992-1995 are included for those wines that are in the sample database.
  • You can add only 15 additional wines and/or tasting notes.
  • The optional 'tutorial' pop-up messages will always display in the demo version.

The file RPDEMO.EXE is self extracting program. You should put it into a temporary directory. Run the program which will uncompress the files to about 5 megabytes in size. One of the files will be README.TXT. Read this text file for instructions on installing the demo software.

The installed software will be about 10 megabytes in size. You may wish to delete RPDEMO.EXE after running it to save disk space. You can delete the entire temporary directory after installing the demo software.
