System Requirements


Wine Technologies recommends that you own a 486 or Pentium PC with at least 8MB of memory (RAM) and 40MB of free hard disk space to operate the software and all available data and indices. The software operates equally well under Windows 3.x and Windows 95.

Note to Macintosh Owners: We have no plans for a Mac version of the software. While a few people are successfully running the software under SoftWindows (http://www.insignia.com) on a Macintosh, there is a significant speed penalty and we can provide no technical support for the Mac.
