Stepping Among Windows Using the Keyboard
To get around the problem of obscured windows using the mouse, click the right hand mouse on: 1) any button on a form, 2) the background of a form (try close to the edge to make sure you're not getting a control), or 3) in the middle of a table of wines.

This will bring up a menu of actions as shown to the left. Selecting Push moves the current window (the one you clicked) behind all other windows.
